## Добавить в Vercel (Settings → Environment Variables)

### 1. TELEGRAM_ADMIN_CHAT_ID
- Напиши в Telegram боту **@userinfobot**
- Он ответит: "Your ID: 123456789"
- Добавь это число как значение

### 2. SUPABASE_WEBHOOK_SECRET
- Придумай любую строку, например: `marathon_secret_2026_xyz`

### 3. GEMINI_API_KEY 🆓 (бесплатно!)
1. Зайди на **https://aistudio.google.com**
2. Войди через Google аккаунт
3. Нажми **"Get API Key"** → **"Create API key"**
4. Скопируй ключ (начинается с `AIzaSy...`)
5. Добавь в Vercel как `GEMINI_API_KEY`

**Лимиты бесплатного плана Gemini:**
- 60 запросов в минуту
- 1500 запросов в день
- Полностью бесплатно, без карты!

## Настройка Supabase Webhook (уведомления о регистрациях)

1. Supabase Dashboard → **Database → Webhooks**
2. Нажми **"Add Webhook"**
3. Заполни:
   - Name: `marathon-notifications`
   - Table: `participants`
   - Events: ✅ INSERT ✅ UPDATE ✅ DELETE
   - URL: `https://твой-сайт.vercel.app/api/supabase-webhook`
   - HTTP Header: `x-webhook-secret` = (значение SUPABASE_WEBHOOK_SECRET)
4. Сохрани

Теперь при каждой регистрации бот напишет тебе в Telegram! 🎉
